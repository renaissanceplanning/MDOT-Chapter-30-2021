Score Insights

Project 22-10 is comprised of long run times and few stops spread over a large project area. Further, while new service is provided between some links, other portions duplicate existing service. These factors likely contributed to lower accessibility impact. The travel time savings estimate shows no benefits for existing riders, as Project 22-10 does not replace or improve existing transit options, and therefore there is no existing ridership.
